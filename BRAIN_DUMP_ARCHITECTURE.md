## Key Design Decisions

1. **Separator Auto-Detection**: Reduces cognitive load on users
2. **Real-time Parsing**: Instant feedback improves UX
3. **Preview Before Create**: Allows review and editing
4. **Batch Processing**: Efficient database operations
5. **Partial Failure Handling**: Graceful degradation
6. **MVI Pattern**: Clean, testable, maintainable architecture
7. **Component Reusability**: BrainDumpParser is standalone
8. **Keyboard Shortcuts**: Power user support
9. **Accessibility**: Screen reader and keyboard navigation
10. **Performance**: LazyColumn for large lists
